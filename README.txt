В качестве менеджера пакетов использовалась утилита Composer. 
Для хранения данных использовалась база данных MySQL 5.7+.
Необходимо наличие языка программирования PHP 8.0+.
Для использования данного ПО необходимо
1) запустить локальный MySQL сервер и создать схему базы данных "laravel";
2) в папке проекта запустить консольные команду php artisan migrate:fresh;
3) запусть локальный веб-сервер консольной командой php artisan serve;
4) запустить фронтенд консольной командой nmp run dev.
